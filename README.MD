# pokergames-cli

_Note: this entire project was just an exercise to practice my python chops, it isn't perfect_

### Prerequisites

```
pip3 install -r requirements.txt
```

### How to use

```
python3 main.py {name of venue}
```

### Example

```
python3 main.py hollywood stl poker
```
